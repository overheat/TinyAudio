# TinyAudio
#### Audio IN/OUT accessory for tiny PC.
- Based on STM32F4Cube.
- Generated by STM32CubeMX.
- Take ST cube demo as reference.

## Requirement

### Hardware 
- Nucleo-F401
- X-Nucleo-CCA01M1(Audio amplifier)
- X-Nucleo-CCA02M1(MEMS microphone)
- 5~20V power source
- 2 loud speaker

### Software

- IAR 8
- STM32CubeMX
